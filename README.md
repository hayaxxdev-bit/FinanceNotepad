## 🎯 Kenapa Aplikasi Ini Ada?
Kalau kamu malas membuka Excel, pusing melihat rumus, atau merasa aplikasi keuangan di PlayStore terlalu ribet, aplikasi ini adalah jawabannya. 

Tugasmu cuma masukin angka, biarkan sistem kami yang menghitung sisanya. Kamu tinggal melihat hasilnya dan memutuskan apakah hari ini bisa makan enak atau harus makan mie instan lagi.

## ✨ Fitur Utama
*   💰 **Pemasukan (Gaji):** Catat total cuan yang masuk ke rekening.
*   💸 **Pengeluaran:** Pantau kemana saja duitmu menguap.
*   🏦 **Biaya Admin:** Potongan receh dari bank/e-wallet yang kalau digabung bikin nangis, catat di sini.
*   🐷 **Tabungan:** Alokasi untuk masa depan (kalau masih ada sisa).
*   🧮 **Kalkulator Otomatis:** Menghitung instan dengan rumus Saldo Bersih = Pemasukan - (Pengeluaran + Biaya Admin + Tabungan).

## 🛠️ Persyaratan Sistem
Pastikan kamu sudah menginstal ini di komputermu:
*   [Node.js](https://nodejs.org/) (Minimal versi 16.x)
*   NPM (Sudah bawaan dari Node.js)
*   [Vercel CLI](https://vercel.com/cli) (Untuk *deploy* ke internet)
